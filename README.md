# HOCR

Projeto acadêmico para prática de versionamento e colaboração em equipe, compartilhando destinos de viagem dos sonhos.

---

## 📖 Sobre o Projeto

O **HOCR** é um projeto desenvolvido para apresentar **lugares que gostaríamos de visitar ao redor do mundo**, reunindo informações como motivos para conhecer esses destinos, curiosidades e aspectos interessantes sobre cada local.

A proposta é praticar o desenvolvimento web básico e a colaboração em equipe utilizando **Git e GitHub**, simulando um fluxo real de desenvolvimento com criação de branches, pull requests e revisão de código, enquanto compartilhamos inspirações de viagem.

---

## 🎯 Objetivos

- Praticar criação e gerenciamento de branches
- Aplicar fluxo de Pull Requests
- Exercitar revisão de código
- Trabalhar em equipe
- Utilizar Markdown para documentação
- Desenvolver páginas web com HTML, CSS e JavaScript

---
